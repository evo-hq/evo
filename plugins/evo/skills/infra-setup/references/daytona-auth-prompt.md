Export a Daytona API key in the shell that launches evo:

```bash
export DAYTONA_API_KEY=...
```

If you need to override the API URL or target, set them explicitly too:

```bash
export DAYTONA_API_URL=https://app.daytona.io/api
export DAYTONA_TARGET=us
```

Then rerun the evo backend switch in the same shell.
